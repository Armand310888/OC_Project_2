# OC_Project_2 – Script Python de traitement de données

## Description
Ce projet est réalisé dans le cadre de la formation *Développeur d’applications Python* chez OpenClassrooms.

Il s’agit d’un script Python permettant de :
- récupérer et extraire les données depuis une page web (HTML),
- effectuer des transformations simples,
- produire un résultat exploitable (affichage ou fichier de sortie).

L’objectif principal est de mettre en pratique les bases du langage Python, la structuration d’un projet et l’utilisation de Git et Github.

## Prérequis
- Python 3.x
- Les dépendances listées dans `requirements.txt` (le cas échéant)

## Installation
1. Cloner le dépôt :
```bash
git clone <https://github.com/Armand310888/OC_Project_2.git>
```
2. Se placer dans le dossier du projet :
```bash
cd OC_Project_2
```
3. Installer les dépendances :
```bash
pip install -r requirements.txt
```

## Utilisation
```bash
python src/main.py
```
